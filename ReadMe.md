#Poker hands sorting

Program is written in C language.

Tested on Ubuntu 20.04 LTS

####Compiling

In folder ./poker run:

`$ make`


####Running

In folder _poker_ run

`$ ./poker`

To feed stream from file run

`$ cat ./test.txt | ./poker`

or

`$ cat ./testOmaha.txt | ./poker --omaha`


####Extra credits:
- implemented option `--omaha`
